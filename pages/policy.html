Політика конфіденційності

Ця Політика конфіденційності описує, як CROSS P CONSULTING (далі – «Ми», «Нас») збирає, використовує та захищає інформацію, яку Ви надаєте, використовуючи наш вебсайт cross-p-consalting.com. Ми цінуємо Вашу довіру та зобов'язуємося захищати Вашу особисту інформацію.

1. Яку інформацію ми збираємо?

Ми можемо збирати такі види особистої інформації:

Прізвище, ім'я: Для ідентифікації користувача.

Номер телефону: Для зв'язку з Вами щодо Ваших замовлень, запитів або послуг.

Посилання на соціальні мережі (Facebook, LinkedIn): Для зручності зв'язку або верифікації, якщо Ви вирішите надати цю інформацію.

Адреса електронної пошти: Для надсилання підтверджень, відповідей на запитання, інформації про замовлення та послуги.

2. Як ми використовуємо Вашу інформацію?

Зібрана інформація використовується виключно для таких цілей:

Обробка замовлень книги: Для виконання Ваших замовлень, доставки книг та зв'язку щодо статусу замовлення.

Написання відгуку про книгу: Для публікації Ваших відгуків на сайті з Вашого дозволу.

Написання питання: Для надання відповідей на Ваші запитання та підтримки комунікації.

Звернення за отриманням послуг: Для обробки Ваших запитів на послуги та забезпечення їх надання.

3. Як ми захищаємо Вашу інформацію?

Ми вживаємо відповідні технічні та організаційні заходи для захисту Вашої особистої інформації від несанкціонованого доступу, зміни, розголошення або знищення. Це включає використання захищених серверів, шифрування даних та обмеження доступу до Вашої інформації лише тим співробітникам, яким вона необхідна для виконання їхніх обов'язків.

4. Розкриття інформації третім сторонам

Ми не продаємо, не обмінюємо та не передаємо Вашу особисту інформацію третім сторонам без Вашої згоди, за винятком випадків, передбачених законодавством, або якщо це необхідно для надання запитуваних Вами послуг (наприклад, службам доставки для замовлень книг).

5. Ваші права

Ви маєте право:

Запитувати доступ до Вашої особистої інформації, яку ми зберігаємо.

Запитувати виправлення будь-яких неточних даних.

Запитувати видалення Вашої особистої інформації.

Відкликати свою згоду на обробку даних у будь-який час.

6. Зміни до цієї Політики конфіденційності

Ми можемо час від часу оновлювати цю Політику конфіденційності. Будь-які зміни будуть опубліковані на цій сторінці. Рекомендуємо періодично переглядати цю сторінку для ознайомлення з останніми змінами.
